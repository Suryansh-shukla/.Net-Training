# React Fake API App

A sample React app built with Vite that fetches data from a fake API and displays it using reusable components.

## Fake API used
- https://jsonplaceholder.typicode.com/users

## Features
- React + Vite setup
- Fetch data with `useEffect`
- Loading state
- Error handling
- Reusable components
- Clean CSS UI

## How to run

```bash
npm install
npm run dev
```

Then open the localhost URL shown in the terminal.

## Build for production

```bash
npm run build
npm run preview
```
