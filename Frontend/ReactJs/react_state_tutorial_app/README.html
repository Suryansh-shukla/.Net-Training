React State Tutorial Project Guide

This project contains tutorial-based React examples for:

  • How to use state in React - simple example
  • How to use state in React - medium example
  • How to use state in React - advanced example
  • How state data can be shared from one component to another
  • How to send data from one component to another using props

How to Run

  1. Extract the zip file.
  2. Open terminal inside the extracted folder.
  3. Run npm install
  4. Run npm run dev
  5. Open the local Vite URL shown in the terminal.

Teaching Notes

  • Simple example: Explain that state stores changing data.
  • Medium example: Explain object state and generic form handlers.
  • Advanced example: Explain arrays, mapping, filtering, toggling, and functional updates.
  • Data sharing example: Explain lifting state up.
  • Send data example: Explain parent-to-child communication through props.
